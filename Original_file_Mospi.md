# Procedure of exportation of data

Install and open Nesstar explorer
Export all data set in format sav
Install pspp
Open file, then nouveau , then syntaxe
Write:

SAVE TRANSLATE
  /OUTFILE="C:\Users\Lenovo\Documents\INDEDATA\block5.csv"  
  /TYPE=CSV
  /FIELDNAMES      
  /CELLS=LABELS.
